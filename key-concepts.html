Glossary

  • Rendering Engine — turns HTML/CSS into pixels on screen.
  • JavaScript Engine — runs JS to add interactivity.
  • Standards — shared rules from W3C/WHATWG for interoperability.
  • Mobile‑First — design for small screens first, then enhance.
  • Progressive Enhancement — start simple, add styles/JS as extras.
  • Viewport — the visible area on a device; set with meta viewport.
  • Cache — stored files to load pages faster.
